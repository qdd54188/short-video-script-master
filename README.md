# 短视频口播脚本生成器

输入主题和类型，自动生成抖音/快手/B站/视频号/小红书全平台口播脚本。

## 功能

- 6 种脚本风格：知识干货、产品种草、故事叙事、观点输出、生活记录、搞笑段子
- 逐秒脚本（台词 + 画面 + 字幕效果 + 字幕位置）
- 拍摄准备清单（服装、道具、灯光、收音）
- B-roll 辅助画面建议
- 5 个爆款视频标题 + 封面方案
- 平台差异适配（抖音快节奏 vs 视频号慢节奏 vs B站深内容）

## 使用

将 SKILL.md 放到 Claude Code 的 `.claude/skills/` 目录下即可使用。
