# 短视频口播脚本生成器

一个 AI Agent Skill（Claude Code / Codex / Cursor 通用），输入主题和类型，自动生成全平台口播脚本。

## 能做什么

- **6 种脚本风格**：知识干货、产品种草、故事叙事、观点输出、生活记录、搞笑段子
- **逐秒分镜脚本**：每一秒的台词、画面切换、字幕位置、字幕特效，精确到秒
- **8 种开头钩子公式**：数字冲击、反问暴击、秘密揭露、结果前置、身份认同…
- **拍摄准备清单**：场景、服装、灯光、收音、道具，全部列好 + 没有某样东西时的替代方案
- **B-roll 辅助画面**：主画面之外需要拍的补充镜头，附带可搜索的关键词
- **平台差异适配**：抖音快节奏、快手接地气、视频号慢节奏、B站有梗有深度、小红书重氛围
- **标题 + 封面方案**：5 个视频标题候选 + 封面图方案
- **结尾引导话术**：不同平台的不同「关注/转发/评论」引导方式

## 安装使用

将 `SKILL.md` 放到 Claude Code / Codex / Cursor 的 skills 目录下即可。

```
Claude Code：~/.claude/skills/
Cursor：.cursor/skills/
Codex：.codex/skills/
```

## 使用示例

```
用户：帮我写一个 60 秒的健康饮食干货脚本，抖音平台，闺蜜人设

输出：
  脚本结构 → 逐秒口播脚本 → 拍摄准备清单 → B-roll建议 → 标题候选 → 拍摄鼓励
```
