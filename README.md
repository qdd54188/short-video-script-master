## 🤔 为什么做这个

做了小红书标题工具之后，很多人问我：**能不能出个视频版的？**

短视频最大的痛不是「不知道拍什么」，而是：

> 脑子里有内容，一开镜头就不知道说什么。磕磕巴巴录了八遍，最后剪出来 30 秒能用。加字幕、配 B-roll，又是半小时。

大神拍短视频有脚本，精确到每一秒该说什么、画面切什么。但大多数人不写脚本——不是不想写，是不会写。

这个 Skill 就是帮你写脚本的那个人。

---

## 🎯 它帮你做什么

```
你说：帮我写一个 60 秒的健康饮食干货脚本，抖音平台，闺蜜人设

它输出：
  ⏱ [0-3秒]   开头钩子，台词怎么写、字幕放哪
  ⏱ [3-8秒]   痛点展开，什么时候切换 B-roll
  ⏱ [8-40秒]  核心干货，怎么分段、怎么留悬念
  ⏱ [40-55秒] 情绪高潮，怎么让观众忍不住点赞
  ⏱ [55-60秒] 结尾引导，怎么让人关注你

  🎬 拍摄清单：穿什么、在哪拍、用什么光、需要什么道具
  📦 B-roll 建议：哪些辅助画面需要补拍
  🏷️ 5 个视频标题 + 封面方案
```

**不是给你一段文字让你照着念。是给你精确到秒的分镜脚本。**

## ⚡ 快速开始

把 `SKILL.md` 复制到你的 AI 编码工具 skills 目录：

```
# Claude Code
cp SKILL.md ~/.claude/skills/

# Cursor
cp SKILL.md .cursor/skills/

# Codex
cp SKILL.md .codex/skills/
```

然后对 AI 说：**「帮我写短视频脚本，XX类型，XX内容，XX平台」**

## ❓ 常见问题

<details>
<summary><b>我面对镜头紧张，能用吗？</b></summary>
能。Skill 会根据你的人设（闺蜜/专业/幽默/温柔）调整语气的自然度，还会在脚本里加自嘲桥段帮你缓解：「第一次录有点紧张，你们凑合看」——观众反而更喜欢真实的你。
</details>

<details>
<summary><b>我没有专业设备怎么办？</b></summary>
你的手机够了。拍摄清单会标注哪些需要专业设备，哪些手机就能搞定。如果某样东西没有，Skill 会给出替代方案。
</details>

<details>
<summary><b>Pro 版和免费的 AI（ChatGPT/Claude）有什么区别？</b></summary>
通用的 AI 不知道抖音前 3 秒的完播率规则、快手的方言偏好、视频号的受众年龄层、B站的弹幕互动设计。这个 Skill 把这些平台差异内置了。
</details>

<details>
<summary><b>不满意能退吗？</b></summary>
7 天内无条件退款。爱发电私信我就行。
</details>
